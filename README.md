# Candle Store Website

This is a full-stack e-commerce website built for a candle business. The platform allows customers to browse products, view details, and place orders, while providing the business owner with an admin system to manage products and inventory.

## 🧾 Purpose

The goal of this project is to provide a functional online store for a real business, enabling product visibility, customer interaction, and future online sales expansion.

## 🧱 Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
- Prisma ORM

### Services
- Cloudinary (image storage)
- Stripe (payment integration - optional / test mode)

## 🛍️ Features

### Customer Side
- Product browsing
- Product search
- Shopping cart
- Product details page
- Contact form

### Admin Side
- Add / edit / delete products
- Upload product images
- Manage inventory
- View orders (if enabled)

## 🚀 Status

In active development for a live business deployment.
